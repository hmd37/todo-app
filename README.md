## 🚀 **Features**

- User authentication (Login, Register)
- Create and manage multiple To-Do lists
- Calendar view for task scheduling

## 🛠️ **Technologies Used**

- Django (Python Web Framework)
- SQLite (Database)
- HTML/CSS
- AJAX (Asynchronous JavaScript and XML)
